PRODIGY_DS_04
Social Media Sentiment Analysis

This project is part of my Prodigy InfoTech Data Science Internship (Task 04).

In this task, I analyzed social media tweets to understand public sentiment toward different brands and topics. The dataset was cleaned, explored, and visualized to identify sentiment patterns and public opinions.

##Tools Used
 *Python
 *Pandas
 *NumPy
 *Matplotlib
 *Seaborn
 *Google Colab
##Highlights
 *Cleaned and explored the Twitter dataset.
 *Analyzed Positive, Negative, Neutral, and Irrelevant sentiments.
 *Visualized sentiment distribution using different charts.
 *Compared sentiments across multiple topics and brands.
